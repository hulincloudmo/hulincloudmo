# 各类语言通用

# 1.多使用return结束函数

在很多代码书中都推荐一个函数应该只有一个return值，这是为了保证函数出口的统一性，但是我们的函数往往是有很多逻辑判断，很多逻辑分支，为了保存状态，我们还得定义大量的临时变量去保存程序的状态………如果我们能够用return提前结束不可能执行的代码，那不就是对我们思维的减负么？

# 2.少使用大量的if/else判断

大量的if/else的可读性是很差的，其实我们这时候可以用switch/case的，switch/case的可读性是高很多的
