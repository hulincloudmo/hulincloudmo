## jsx语法

### 注释的写法

```typescript jsx
{/*这是一个注释*/}
```

### 引入样式

::: tip
react中引入css文件:
import './App.css'
:::

::: warning
不能在元素中使用class引入样式类，而是应该使用className表示
className='xxx'
:::
