1. `concurrently`

此包可以帮助你同时运行两个文件

2. `cross-env`

跨平台配置环境变量

3. `wait-on`

等待某个服务完成后才启动命令

4. `classnames`

帮助你动态的拼接class

5. `query-string`

解析url中的查询参数，能够自动进行URL编码

6. `vconsole`

方便的查看console输出信息，可配合`AlloyLever`使用

7. `day-js、moment`

日期格式转换

8. `nvm切换默认node版本`

`nvm alias default stable`

9. `numjs转换数据格式`

10. `npm一键更新`

`npm install -g npm-check-updates`

再使用ncu命令即可更新node_modules
