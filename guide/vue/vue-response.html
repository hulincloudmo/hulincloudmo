# vue响应式数据更新

# vue的非响应情况

# 1.直接修改数组内的一个元素的值(利用索引直接设置一个项)或直接修改数组长度

arr[1] = "2"
arr.length = 0

# 2.在一个组件实例中，只有在data里初始化的数据才是响应的，Vue不能检测到对象属性的添加或删除，没有在data里声明的属性不是响应的。

# 为什么会出现这样的现象呢？

解决上面两个非响应式的方法网上有很多，所以我们重点关注，为什么会不能响应式更新呢？ 要实现响应式，实际上就是实现一个观察者，观察者一直在关注着一个对象或者数组的变化。
