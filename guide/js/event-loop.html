# 宏任务与微任务

宏任务与微任务的概念非常重要，需要我们详知，首先我们看看那些属于宏任务，那些属于微任务

宏任务：

  • script

  • setTimeout

  • new promise

  • setInterval

  • I/O

微任务

  • process.nextTick

  • promise的then

  • async、await

js的执行顺序是：执行宏任务，如果宏任务产生了微任务，将会优先执行，当宏任务全部执行完毕后，执行微任务
